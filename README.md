# 🌟 Interactive Animated Dashboard

A modern, highly interactive dashboard that focuses on delivering an exceptional User Experience (UX) through a complex, interconnected series of smooth animations. This project is built to be a visual journey rather than just a data display.

## ✨ Features & Animation Flow (Choreography)

The UI elements are carefully choreographed to appear in a precise sequence:

1. Intro Animation: The main layout enters dynamically from the bottom-left to the top-right.
2. Logo Reveal: A small white circle expands, revealing a petal-shaped logo with a staggered animation (each petal delayed slightly after the previous one).
3. TopBar Expansion: The logo container expands to the right to reveal the site name with a smooth text animation, then moves to the top corner, expanding fully to form the TopBar.
4. Layout Sync: Synchronized with the TopBar, the SideBar and Date containers slide in from the left, gradually increasing in width.
5. Content Fade-in: The date, sidebar tabs, and top bar content appear smoothly from bottom to top with a fade-in effect.
6. Data Visualization: The chart containers appear sequentially, showcasing:
   - A Donut Chart with a custom legend.
   - A Bar Chart with interactive tooltips.
   - A custom-built HeatMap.
   - Animated incrementing number wheels for key metrics.

## 📱 Responsiveness

The dashboard is optimized for various screen sizes to maintain a clean UI:
- SideBar: Hidden on medium/small screens, with a smooth slide-in/out animation triggered by the TopBar hamburger menu.
- Search Bar: The input field is hidden by default and expands smoothly only when the search icon is clicked, saving space on smaller screens.

## 🛠️ Tech Stack

- Vite: Fast frontend build tool.
- React.js: Component-based UI library.
- Tailwind CSS: For rapid, responsive styling.
- Visx: Used for building the complex, low-level SVG HeatMap.
- Chart.js / Recharts: For the Donut and Bar charts.
- Framer Motion: For orchestrating the complex sequence of animations.
- React Datepicker: For seamless date selection.

## ⚠️ Development Status
